# Hour-Based Marks Prediction Using Supervised Machine Learning

## Project Overview

This project uses Supervised Machine Learning to predict student marks based on the number of hours studied. The objective is to analyze the relationship between study time and academic performance and build a predictive model capable of estimating marks from study hours.

The project demonstrates the practical application of predictive analytics and regression techniques in educational data analysis.

---

## Problem Statement

Understanding how study time impacts academic performance can help students and educators make informed decisions. This project aims to predict student scores based on the number of hours studied using machine learning techniques.

---

## Objectives

- Analyze the relationship between study hours and student marks.
- Perform data cleaning and exploratory data analysis (EDA).
- Develop a supervised machine learning model for score prediction.
- Evaluate model performance using regression metrics.
- Visualize trends and prediction results.

---

## Dataset Description

The dataset contains information about:

- Hours Studied
- Student Scores (Marks)

### Target Variable

- Score (Marks)

### Feature Variable

- Hours Studied

---

## Project Workflow

### 1. Data Collection

- Imported the dataset containing study hours and corresponding scores.

### 2. Data Preprocessing

- Checked for missing values.
- Verified data consistency.
- Prepared data for model training.

### 3. Exploratory Data Analysis (EDA)

- Visualized the relationship between study hours and marks.
- Identified trends and correlations.
- Generated plots for better understanding of the dataset.

### 4. Model Development

Implemented a Linear Regression model to predict student marks based on study hours.

### 5. Model Evaluation

Evaluated model performance using:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

### 6. Prediction

- Predicted student scores for given study hours.
- Compared actual vs predicted results.

---

## Key Insights

- A strong positive relationship exists between study hours and student performance.
- Students who dedicate more time to studying generally achieve higher scores.
- Linear Regression effectively models the relationship between study hours and marks.

---

## Results

The Linear Regression model successfully predicted student marks based on study hours and demonstrated the practical use of supervised machine learning in educational performance forecasting.
